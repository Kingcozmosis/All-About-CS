About the Developer

Hey, I’m K1NG a competitive Counter-Strike player and developer.

I built All Things CS because I wanted a single place inside Discord where players could quickly access everything — matches, stats, teams, news, and updates without having to jump between multiple sites.

What started as a small project quickly turned into something much bigger as I kept adding features, improving performance, and making it more useful for everyday CS players.

The Goal

The goal is simple:

Build the best all-in-one Counter-Strike Discord bot.

Whether you're checking live matches, comparing players, or just staying updated with CS2 news, the bot is designed to make everything fast, clean, and easy to use.

What Makes It Different

• Clean, interactive UI inside Discord

• Real time data powered by BO3.gg

• Constant updates and new features

• Built by an actual CS player (not just a random dev)

Future Plans
